# Bubble_Panel

Модульная система управления «пузырьками» (операционными промптами).

## 📁 Структура бордов

- control     — управление форматом и режимами вывода
- dialogue    — режимы расспроса и взаимодействия
- execution   — инициирование действий
- guard       — ограничения и нормативные режимы
- meta        — анализ системы
- system      — архитектурные основы
- transform   — преобразование формата данных
