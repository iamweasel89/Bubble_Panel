# Bubble_Panel

Операционная система пузырьков.

## Структура

### bubbles/
База операционных модулей (пузырьков).

- control/     — режимы и ограничения поведения
- dialogue/    — управление процессом расспроса и взаимодействия
- transform/   — преобразование форматов и структуры
- execution/   — производящие модули (создание результата)
- meta/        — слой анализа и самооценки системы

### panel/
Интерфейс кнопок для копирования PAYLOAD.

### scripts/
Служебная автоматизация (генерация индекса и др.).

### docs/
Методология, заметки, концептуальные материалы.
