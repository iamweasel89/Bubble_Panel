# 🧠 PAYLOAD

Режим: Структурная диагностика изображения.

Назначение:
Универсальная классификация базовой структурной организации изображения
перед любым дальнейшим анализом.

Не анализировать:
- смысл
- назначение
- стиль
- материал
- символы как смысл
- производственный процесс

Вход:
Любое изображение.

Вывод строго по структуре:
--------------------------------------------------
1. СТРУКТУРНЫЕ ИНДИКАТОРЫ
1.1 Наличие центральной доминанты:
- да / нет / невозможно определить
1.2 Наличие автономных визуально отделённых элементов:
- да / нет / невозможно определить
1.3 Наличие регулярной сеточной или модульной организации:
- да / нет / невозможно определить
1.4 Тип фонового пространства:
- равномерная заливка
- равномерная заливка + вспомогательные линии
- геометрический орнамент
- текстурный фон
- сложная пространственная сцена
- невозможно определить
1.5 Присутствуют ли направляющие / осевые / вспомогательные линии,
которые НЕ являются частью основных форм:
- да / нет / невозможно определить
--------------------------------------------------
2. НЕОПРЕДЕЛЁННОСТИ
Указывать только неопределённости, связанные с пунктами 1.1–1.5.
--------------------------------------------------
3. ДОМИНИРУЮЩИЕ ПРИНЦИПЫ ОРГАНИЗАЦИИ
- центрирование
- симметрия
- сетка
- повтор формы
- свободная композиция
- невозможно определить
--------------------------------------------------
4. УРОВЕНЬ УВЕРЕННОСТИ
Оценка по каждому подпункту 1.1–1.5 в процентах.
--------------------------------------------------
Ограничения:
- Если признак нельзя определить строго визуально — указать "невозможно определить".
- Не добавлять дополнительные разделы.
- Не выходить за пределы структуры.

Роль в пайплайне:
Фаза 0. Обязательный входной фильтр перед геометрическим анализом.
