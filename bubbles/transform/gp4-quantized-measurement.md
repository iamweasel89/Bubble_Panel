# 🧠 PAYLOAD

Работай по протоколу GP-4 (квантованный измерительный режим, шаг 5%).

ПРИНЦИП:
Все процентные значения кратны 5%.
Используется округление до ближайших 5%.
После округления проверяется формула согласованности.

БЛОК 0 — ПРАВИЛА ИЗМЕРЕНИЯ (обязательны)

Главный объект — крупный кольцевой объект слева.
Bounding box внешнего круга = 100%.
Внешний диаметр всегда = 100%.
Внутренний диаметр измеряется по границе отверстия.

Толщина рассчитывается строго по формуле:
Толщина = (100 − Внутренний диаметр) / 2

Все значения округлять до ближайших 5%.

После округления проверить:
Внутренний диаметр + 2 × Толщина = 100

Если формула не сходится — скорректировать толщину.

Запрещено:
- использовать проценты не кратные 5
- добавлять текст вне структуры

БЛОК 1 — ИЗМЕРЕНИЯ (квантованные)

1. Внешний диаметр: ___ %
2. Внутренний диаметр: ___ %
3. Толщина кольца (расчётная): ___ %
4. Высота объекта с выступами: ___ %
5. Количество радиальных сегментов: ___
6. Угол между сегментами: ___ °

БЛОК 2 — СИММЕТРИЯ

1. Вертикальная ось симметрии:
- есть
- нет

2. Горизонтальная ось симметрии:
- есть
- нет

3. Радиальная симметрия:
- полная
- частичная
- отсутствует

БЛОК 3 — ПРОВЕРКА СОГЛАСОВАННОСТИ

Контрольная формула:
Внутренний диаметр + 2 × Толщина = ___ %

(должно быть 100%)

БЛОК 4 — НЕОПРЕДЕЛЁННОСТИ

Заполняется только если невозможно соблюсти правила квантования.

Строго соблюдать структуру.
Никаких комментариев вне блоков.
